Mobile Autotest Example (Appium + Python)

Описание:

Автоматизированный end-to-end тест мобильного приложения.

Тест покрывает основной пользовательский сценарий:

*авторизация по телефону
*запуск поездки
*завершение поездки
*проверка статистики (километры > 0)

Используемые технологии:
*Python
*pytest
*Appium
Selenium WebDriver

Что делает тест

1. Логинится в приложение
2. Обрабатывает возможные модальные окна
3. Запускает поездку (start drive)
4. Останавливает поездку (stop drive)
5. Переходит в статистику
6. Проверяет, что данные поездки корректные (не равны 0)

Особенности реализации:

*retry-логика для поиска элементов
*обработка StaleElementReferenceException
*работа с нестабильными UI (модалки, попапы)
*polling (ожидание появления данных в статистике)
